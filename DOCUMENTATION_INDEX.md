## 🚀 QUICK START COMMAND

Instead of reading, just follow these commands:

```bash
# 1. Get API key from https://astronomyapi.com/

# 2. Edit backend/src/main/resources/application.properties
#    Set: astronomy.api.key=YOUR_KEY

# 3. Terminal 1 - Backend
cd backend
mvn spring-boot:run

# 4. Terminal 2 - Frontend
cd frontend
npm install
npm run dev

# 5. Open browser
#    http://localhost:3000

# 6. Fill form and click "Generate Star Chart"
# Done! 🎉
```

For more details, see **SETUP.md**.
